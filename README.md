# Programmatic Screenshots of Any Website in Seconds using Next.JS & Puppeteer by Themeptation

## How to use it?

Execute to install dependecies

```bash
npm install
# or
yarn
```

if it's done execute

```bash
npm run dev
# or
yarn dev
```

Open your browser and type https://localhost:3000
